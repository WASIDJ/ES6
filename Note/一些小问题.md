# 解构赋值
## 1
```js
function move({x, y} = { x: 0, y: 0 }) {
  return [x, y];
}
move({x: 3, y: 8}); // [3, 8]
move({x: 3}); // [3, undefined]
move({}); // [undefined, undefined]
move(); // [0, 0]

// 正确例子
function move({x = 0, y = 0} = {}) {
  return [x, y];
}
move({x: 3, y: 8}); // [3, 8]
move({x: 3}); // [3, 0]
move({}); // [0, 0]
move(); // [0, 0]
```
- Q：解释为什么？
  - A：因为函数参数的解构赋值默认值是一个对象，所以当参数没有传递值时，会使用默认值。
  - B：空对象时，对`{x: 3, y: 8}`解析
